MsgQueue<LogMsg*> external_logs
static

This queue is used for receiving logs from external sources. No adaptive processing is done to them, as the senders should do that before forwarding the logs over the network. This is usually populated by the SaveLogMsg Logic request, and whatever is written to this queue will be dumped into our log file directly.
