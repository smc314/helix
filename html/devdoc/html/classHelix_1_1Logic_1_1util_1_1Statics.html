void Statics::ReplaceSessionVars ( IOConn &  ioc,
twine &  target 
)
static

This method handles replacing session variables in the given string. The string should have tokens in the form of ${VarName} embedded within it, and we'll parse them out, look up "VarName" in our current session settings and, if found, replace the token in the string with the value from the session settings.
