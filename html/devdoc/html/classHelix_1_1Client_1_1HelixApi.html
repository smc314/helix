Detailed Description

This API represents the C++ interface to the Hub server. All API's that are available to the Javascript client are also available here, and re-generated each time the Hub is compiled. Do Not Edit This Class directly. Update the code generator and make your edits there.

char * HelixApi::Get ( const twine &  requestPath)
virtual

Sends a GET request to the Hub and does no response checking - use this for requesting static content from the Hub such as web pages, images, etc. Note, this is an override of the HttpClient::Get method, where we change the understanding of the input requestPath. We do not require it to be a full URL, but will build the URL based on the input host/port setup when creating this HelixApi object.
