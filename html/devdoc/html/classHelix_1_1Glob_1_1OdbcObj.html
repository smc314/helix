Detailed Description

This class defines our interaction with the Odbc database. It encapsulates all of the ODBC calls into a standard object interface. It has everything we need to start from, and we will evolve it as necessary as time goes on.

void OdbcObj::ValidateStmt ( char *  stmt,
int  type 
)
virtual

This function is used to validate the syntax of a statement. It is equivalent to a call to SetStmt followed by a call to SQLNumResultCols. The call to SQLNumResultsCols forces the driver to really evaluate the statement that we prepared and generate any errors that might be necessary. Not all drivers will fully parse and syntax check a statement during the call to SQLPrepare.
