This class is the scaler driver that handles all Processor threads. All it does is handle the scaling.

void MsgProcScaler::ShutdownRestart ( void  )

This allows one of our child threads to tell us to shutdown for a restart only. This isn't quite a full stop. All of the threads will be shutdown and destroyed, and then we will cycle once and start 'em all back up.
